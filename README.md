# Fit Analyzer

> 옷 사이즈 고민을 줄여주는 핏 분석 서비스

🔗 **배포 링크**: https://2026-spring-seven.vercel.app/body

---

## 프로젝트 개요

쇼핑몰 상품 링크와 내 신체 정보를 입력하면,  
옷의 실측 사이즈와 내 신체 치수를 비교해 항목별 핏을 알려줌

---

## 주요 기능

### 1. 신체 치수 추정
키·몸무게·성별을 입력하면 아래 7가지 부위의 치수를 자동 추정

| 부위 | 설명 |
|------|------|
| 어깨 (Shoulders) | 양쪽 어깨 끝 사이 너비 |
| 가슴 (Chest) | 겨드랑이 아래 가슴 둘레 |
| 허리 (Waist) | 가장 좁은 허리 둘레 |
| 엉덩이 (Hip) | 가장 넓은 엉덩이 둘레 |
| 허벅지 (Thigh) | 한쪽 허벅지 둘레 |
| 소매 (Sleeve) | 어깨 끝에서 손목까지 |
| 총장 (Length) | 어깨에서 발목까지 |

> Anthropic Claude API 연동 시: 전신사진 + 기준물체(A4 용지, 500ml 페트병 등 실제 크기를 아는 물체)를 함께 촬영해 업로드하면, 기준물체의 실제 크기를 기준으로 사진 속 신체 비율을 계산해 더 정확한 치수를 추정합니다.

---

### 2. 실측 파싱 및 상품 정보 제공
- 상품 페이지 HTML을 직접 파싱해 사이즈표, 제품명, 상품 이미지를 자동 추출
- 결과 페이지에서 상품 이미지 갤러리 및 제품명 표시
- 라쿠텐 착용샷 검색 링크 자동 제공 (실착 이미지 참고용)

---

### 3. 핏 판정
옷 실측과 신체 치수의 차이(여유분)를 기준으로 4단계로 판정합니다.

| 판정 | 기준 |
|------|------|
| TIGHT | 신체 치수 > 옷 실측 (여유 없음) |
| FIT | 차이 0~3cm |
| REGULAR | 차이 3~6cm |
| LOOSE | 차이 6cm 초과 |

> ※ 모드맨(mode-man.com) 실측 기준 적용: 옷을 바닥에 펼쳐 단면을 재는 방식이므로, 가슴·허리·허벅지 등 둘레 항목은 ×2 변환 후 비교합니다.

---

### 4. 사용자 데이터 저장 및 불러오기
- 이름을 키로 신체 정보(키·몸무게·성별·전신사진)를 로컬에 저장
- 재방문 시 이름 입력만으로 기존 데이터 자동 불러오기

---

## 기술 스택

- Next.js (App Router) · TypeScript · Tailwind CSS
- Anthropic Claude API (사진 기반 체형 분석)
- Vercel 배포

---

## 개발 과정

1. 신체 치수 추정 로직 설계 (`lib/bodyEstimate.ts`)
2. 쇼핑몰 HTML 파싱으로 실측·제품명·이미지 추출 (`lib/modoodmanParser.ts`)
3. 실측 vs 신체 치수 비교 및 핏 판정 (`lib/sizeMatch.ts`)
4. 결과 페이지 UI 구현 (상품 갤러리 + SVG 실루엣 + 수치 비교 도표)
5. 사용자 데이터 저장/불러오기 구현 (localStorage)
6. Anthropic API 연동으로 기준물체 기반 사진 분석 기능 추가
